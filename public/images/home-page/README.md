## Image Guidelines

### Format Recommendations:
- **Logos**: PNG with transparent background
- **Photos**: JPG for photographs
- **Icons**: SVG or PNG with transparent background

### Size Optimization:
- Compress all images before uploading
- Use tools like TinyPNG or ImageOptim
- Target file size: < 200KB per image

### Naming Convention:
- Use lowercase letters
- Use hyphens instead of spaces
- Be descriptive but concise

### Placeholder Images:
Until you add your own images, you can use placeholder services:
- https://placeholder.com/
- https://unsplash.com/ (free stock photos)
- https://picsum.photos/ (random placeholder images)

## How to Add Images

1. Place all images in this `public/images/` folder
2. Reference them in components using `/images/filename.jpg`
3. Ensure file names match exactly what's referenced in the code
4. Test the website after adding images to ensure they load correctly
